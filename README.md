# MASTER THESIS 
Title: Spatial Data Visualization: Assessing Financial Decision Accuracy in Immersive Environments 

## Overview
This application is part of an Immersive Analytics (IA) research project. It aims to evaluate and compare how a 3D Desktop Application vs. a Virtual Reality (VR) Environment affects analytical reasoning and financial decision-making.
The framework utilizes a tree metaphor for stock market data visualization, combining Python for data processing and Unity (C#) for real-time rendering. A comprehensive user evaluation case study was conducted to investigate how immersion enhances data analysis compared to traditional methods.
**Key Concepts**: Immersive Analytics (IA), Data Visualization, Metaphor Visualization, Human Computer Interaction (HCI)

### First Version: 3D Desktop Version - App Functionality 
- Time Navigation
- Multilpe cameras
- Mini Map
- Data Chart
